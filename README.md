practice_skrollr
=====================
這是一個為了練習而亂掰的音樂家or作曲家的網站，內容請無視。
以下是使用的套件：
  - jquery-1.11.0.min.js
  - skrollr.min.js

>雖然這時代說起這件事有點慚愧，但是這專案只能相容於電腦噢。


### Todo's
